# Logo Karamba (PDFs)

Colocá aquí el logo oficial para que aparezca en:

- Certificados de cursos (`pdf-lib`)
- Facturas de pedidos (admin)

**Archivo recomendado:** `karamba-logo.png` (fondo transparente) o `karamba-logo.jpg`.

También se aceptan `logo.png` o `certificate-logo.png` como respaldo.

Opcional: variable de entorno `KARAMBA_LOGO_PATH` con ruta absoluta a un `.png` / `.jpg`.
